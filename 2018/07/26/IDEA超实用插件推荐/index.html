IDEA超实用插件推荐

发表于 2018-07-26 | 更新于 2018-07-28 | 评论数： | 阅读次数：

善用IDEA快捷键和插件，对提高开发效率有很大帮助。下面介绍几款超实用的插件。

人工智能辅助编程插件

  1. Codota
    官网： https://www.codota.com
    插件地址：https://plugins.jetbrains.com/plugin/7638-codota
    安装步骤：https://www.codota.com/plugins/download
    基于AI的代码智能提示工具，Codota给出的联想代码是基于成千上万的开发者的开源代码学习，通过人工智能算法计算而来。
    除了给出的强大代码联想之外，你还可以通过两次点击CTR键或者右键选择“show relevant examples”查看相似代码。
  1. Ai Predictive Coding
    插件地址：https://plugins.jetbrains.com/plugin/9203-ai-predictive-coding
    [1] AI可以预测和你当前调用相关性最高的函数，并将其在联想列表中置顶。
    [2] 可以告诉AI让它帮你做一些小功能如 /// convert myString to int

自动生成

  1. Generate All setters
    官方地址：https://plugins.jetbrains.com/plugin/9360-generateallsetter
    在新的函数中，
    [1] 直接在函数体中使用alt+enter快捷键，选择“generate all setter”,自动创建返回值类型相同的对象并调用所有的set方法，并且会根据参数中的实体名填充，如果没有填充默认值。
    [2] 在某个实体对象名称中使用快捷键，选择该功能，则会生产该实体的所有set的方法，并赋默认值。

  2. Auto filling Java call arguments
    官方地址：https://plugins.jetbrains.com/plugin/8638-auto-filling-java-call-arguments
    在调用函数时，通过alt+enter键，触发联想功能，选择”auto fill call parameters”自动填充调用的函数的形参名。

  3. GenerateO2O
    插件地址：https://plugins.jetbrains.com/plugin/7171-generateo2o
    此插件可以实现一个对象转换为另外一个对象

  4. Auto Transfer
    和插件3类似，自动转换
    本工具已支持List/ArrayList/Set/HashSet/Map/HashMap的转换
    插件地址：https://plugins.jetbrains.com/plugin/8260-auto-transfer

  5. Builder Generator
    自动创建Builder对象

实用工具

  1. Grep Console
    官方地址：https://plugins.jetbrains.com/plugin/7125-grep-console
    [1] 修改符合条件的日志颜色
    [2] 将搜索结果输出到新的控制台
    [3] 过滤掉所有不需要的日志
    等
  2. Mybatis Log Plugin
    官方地址：https://plugins.jetbrains.com/plugin/10065-mybatis-log-plugin
    [1]mybatis 输出的sql日志还原成完整的sql语句。
    [2]将日志输出的sql语句中的问号 ? 替换成真正的参数值。
    [3]通过 “Tools -> MyBatis Log Plugin” 菜单或快捷键 “Ctrl+Shift+Alt+O” 启用。
    [4]点击窗口左边的 “Filter” 按钮，可以过滤不想要输出的sql语句。
    [5]点击窗口左边的 “Format Sql” 按钮，可以格式化输出的sql语句。
    [6]选中console的sql日志，右击 “Restore Sql from Selection” 菜单可以还原sql语句。
    前提条件：输出的sql日志必须包含”Preparing:”和”Parameters:”才能正常解析。
  3. RestfulToolkit
    一套 RESTful 服务开发辅助工具集。
    [1].根据 URL 直接跳转到对应的方法定义 ( Ctrl \ or Ctrl Alt N );
    [2].提供了一个 Services tree 的显示窗口;
    [3].一个简单的 http 请求工具;
    [4].在请求方法上添加了有用功能: 复制生成 URL;,复制方法参数…
    [5].其他功能: java 类上添加 Convert to JSON 功能，格式化 json 数据 ( Windows: Ctrl + Enter; Mac: Command + Enter )。
    支持 Spring 体系 (Spring MVC / Spring Boot)
    支持 JAX-RS
    支持 Java 和 Kotlin 语言。
